# OJT Timetracker - PHP base time logging project

## This PHP based project is a minimalist, very simple project solution for monitoring intern attendance and project hours.


This PHP-based project is designed with a simple, minimalist architecture. It serves as a foundational stepping stone for interns to build their technical skills and gain hands-on experience.

## 🏗️ Project Structure

📂 OJT-Timetracker  
├── 📂 includes/   # Core PHP logic and backend processing  
└── 📂 views/      # Front-end templates and page layouts
