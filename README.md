# Timetracker: A Minimalist PHP based Time Logging System

## A streamlined, functional web application built with PHP and MySQL designed to track and manage internship hours.

This project focuses on a minimalist clean user interface and secure data handling to provide a reliable tool for logging intership progress. Every part of this project is describe below:

* index.php - this is the landing page where everybody as a first step needs to login.
* includes/ folder - this is where all the php files about the project's logic goes to.
* view/ folder - this is where all the files that is all about displaying data / view goes to.
